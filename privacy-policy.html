Política de Privacidade

Última atualização: 2026-05-18

Esta Política de Privacidade descreve como a ferramenta Legendas IA, oferecida pela comunidade Maestria da IA, coleta, utiliza e protege os dados pessoais dos seus usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

1. Dados Coletados

Coletamos apenas os dados estritamente necessários para o funcionamento da ferramenta:

  • E-mail: para verificação de membros aprovados.
  • Telefone (WhatsApp): para verificação de identidade e envio de código OTP.
  • Eventos de uso: ações anônimas dentro da ferramenta (ex.: aberturas, cliques) para melhoria contínua.

Chaves de API de terceiros (OpenAI, Anthropic, etc.) são armazenadas apenas no navegador do usuário (localStorage) e nunca são transmitidas para nossos servidores.

2. Finalidade do Tratamento

  • Verificação de acesso: confirmar que o usuário é membro aprovado da comunidade.
  • Melhoria da ferramenta: analytics anônimos para entender padrões de uso.
  • Comunicação: envio de código OTP via WhatsApp para autenticação.

3. Base Legal

O tratamento de dados é realizado com base no legítimo interesse (Art. 7º, IX da LGPD) para verificação de acesso e na execução de contrato (Art. 7º, V) referente à assinatura da comunidade Maestria da IA.

4. Compartilhamento de Dados

Seus dados não são vendidos ou compartilhados com terceiros para fins comerciais. Utilizamos os seguintes serviços técnicos:

  • Supabase: banco de dados para verificação de membros (hospedado na AWS, dados criptografados em trânsito e em repouso).
  • Netlify: hospedagem da aplicação web.
  • Z-API: envio de mensagens WhatsApp para autenticação OTP.

5. Período de Retenção

  • Dados de verificação: mantidos enquanto o usuário for membro ativo da comunidade.
  • Códigos OTP: expiram em 10 minutos e são limpos automaticamente.
  • Eventos de uso: mantidos por até 12 meses.

6. Seus Direitos (LGPD)

Você tem direito a:

  • Confirmação da existência de tratamento de dados;
  • Acesso aos seus dados pessoais;
  • Correção de dados incompletos ou desatualizados;
  • Eliminação dos dados pessoais (veja o link abaixo);
  • Informação sobre compartilhamento de dados;
  • Revogação do consentimento.

7. Segurança

Implementamos medidas técnicas para proteger seus dados:

  • Comunicação criptografada (HTTPS/TLS);
  • Row-Level Security (RLS) no banco de dados;
  • Limitação de taxa de requisições;
  • Chaves de API nunca armazenadas no servidor.
